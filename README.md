# 基于Rect开发H5模版
### 运行依赖
  - node v13.10.1
  - Mac、Windows、Linux
### 运行脚本
  - lint 代码检测
  - dev mac 开发
  - build:test mac 测试编译
  - build mac 生产编译
  - uw:dev Windows、Linux 开发
  - uw:build:test Windows、Linux 测试编译
  - uw:build Windows、Linux 生产编译
  - analyzer:prod 依赖分析

### 工程结构
 - config webpack打包配置文件
 - dist 构建目录
 - node_modules npm依赖
 - src 源文件
  - assets 静态资源
  - pages 页面
  - public 公共类
  - app.{tsx,scss} 入口
  - index.html index
 ...
